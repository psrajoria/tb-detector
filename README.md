## 📌 About This Project

**TBScan AI** is a personal end-to-end machine learning project built by **Pankaj Rajoria**
that detects Tuberculosis (TB) from chest X-ray images. It combines computer vision,
explainable AI, and a Retrieval-Augmented Generation (RAG) pipeline to deliver not just
a prediction — but a visual explanation and structured medical guidance grounded in real
WHO treatment guidelines.

The project will be fully containerised with Docker, deployed free on HuggingFace Spaces, and
accessible to anyone in the world via a simple web link with zero infrastructure cost.

## ✨ Features

- 📤 **Image Upload** — Drag and drop or browse to upload a chest X-ray (JPG / PNG)
- 🤖 **Deep Learning Detection** — VGG16 fine-tuned on 4,200+ labelled TB X-rays
- 🌡️ **Grad-CAM Heatmap** — Visual overlay highlighting suspicious lung regions in the X-ray
- 📊 **Confidence Chart** — Interactive Plotly bar chart showing probability per class
- 💊 **RAG Medical Advisory** — LangChain + ChromaDB + LLaMA 3 (Groq) generates WHO-backed treatment guidance including the standard RIPE regimen
- 🔴 **Risk Indicator** — Colour-coded result (🟢 Normal / 🔴 TB Suspected)
- 🔒 **Secure Configuration** — All API keys stored as environment secrets, never in code
- 📱 **Accessible Anywhere** — Works on desktop, tablet, and mobile from a single URL

## 🗺️ How It Works

User uploads chest X-ray image
↓
Streamlit UI → HTTP POST → FastAPI /predict
↓
┌──────────────────────────────────────────────┐
│ FastAPI Pipeline │
│ │
│ Step 1 → Preprocess image (224×224, norm) │
│ Step 2 → VGG16 inference → label + score │
│ Step 3 → Grad-CAM → heatmap (base64) │
│ Step 4 → ChromaDB → top 4 WHO chunks │
│ Step 5 → Groq LLaMA 3 → advisory text │
└──────────────────────────────────────────────┘
↓
Streamlit displays result:
┌─────────────┬──────────────┬──────────────┐
│ Original │ Grad-CAM │ Confidence │
│ X-Ray │ Heatmap │ Bar Chart │
└─────────────┴──────────────┴──────────────┘
💊 WHO-backed Medication Advisory

# 2. Create your .env file
